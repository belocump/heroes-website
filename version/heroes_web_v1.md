# 「週末の充実感」を裏切らないWEB構成案

## 1. 外部予約サービスへの「迷わせない」導線設計

外部の予約ページに飛ばす際、ユーザーが「あれ？どこのページに行けばいいの？」と迷うのが一番の離脱原因になります。

- **「今週末のイベントを予約する」ボタンの常設**
- スマホ画面の下部に、常に予約ページ（外部URL）へジャンプするボタンを固定（フローティングバナー）します。

- **イベントごとの直リンクボタン**
- WEBサイト上で「ソサイチ」「個サル」の紹介を見たら、そのすぐ下に「【外部サイト】このイベントのスケジュールを見る」というボタンを配置し、ダイレクトに対象の予約ページへ遷移させます。

## 2. メインコンテンツ：週末の選べる3大イベント

「雨の心配がない屋内」という強みを活かし、金・土・日に開催されるバラエティ豊かなイベントをカード型（ボックス型）のデザインで並べます。30〜40代のサッカー好きの「その日の気分や体力」に合わせて選べるようにするのがポイントです。

| イベント名 | ターゲット・魅力の伝え方 | 開催イメージ |
| ---------- | ------------------------ | ------------ |

| **大人の個サル**<br>

<br>（金・土・日） | **「1週間のストレスを、最高のパスワークで発散」**<br>

<br>スタッフも一緒に混ざり、テンポの良いゲームメイクをサポート。仕事終わりの華金や、週末のルーティンに。 | 金曜夜 / 土日 |
| **ソサイチ（少人数制サッカー）**<br>

<br>（土・日） | **「フットサルじゃ物足りない。でも11人は集まらないあなたへ」**<br>

<br>屋内の広いコートを活かし、サッカーに近いロングパスや展開力を楽しめる、大人のためのミニサッカー。 | 土日の日中 |
| **ウォーキングフットボール**<br>

<br>（土・日） | **「走らないから、センスが光る。怪我なく楽しむ究極の頭脳戦」**<br>

<br>「体力的に走るのはキツイ、でもボールは蹴りたい」「怪我は絶対できない」という30〜40代に最適。走戦術やパスの質だけで勝負する新しいスタイル。 | 土日の午前など |

## 3. 「スタッフ共闘」を可視化するコンテンツ

30〜40代は「コミュニティの質」を重視します。スタッフの顔が見えることで、初めての参加でも一気にハードルが下がります。

### 【今週のゲームキャプテン（スタッフ紹介）】

> 「今週末のソサイチを担当する〇〇です！僕が皆さんの良さを引き出すパスを供給します。1人参加の方もチームに馴染めるよう全力でアシストするので、安心してご来場ください！」

💡 **ポイント：** このようなスタッフ紹介と、「雨天中止がないから、確実に週末の予定が確定する！」という安心感をセットでアピールします。

## 🎨 制作に向けた最後のチェックポイント

これで全体のデザインやテキストの方向性は完璧に固まりました！実際にWEBサイトの制作（ワイヤーフレーム作成など）に入る前に、以下だけ事前に決めておくとスムーズです。

- **イベントの「写真・動画」の準備**
- 特に「ウォーキングフットボール」や「ソサイチ」は、言葉だけだとイメージが湧きにくい場合があります。「走ってはいけないルールで、大人が笑顔で楽しんでいる写真」や「ソサイチの広いコート風景」など、実際のイメージに近い素材（フリー素材でも可、できれば実際の様子）をいくつか用意しておくと、デザインに説得力が出ます。

- **キャンセル規定の明記**
- 「外部のフリー予約サービスを使う」とのことですので、WEBサイト側にも「キャンセルは○日前まで無料（外部サイトよりお手続きください）」といった案内を分かりやすく載せておくと、30〜40代の忙しいビジネスパーソンも安心して予約しやすくなります。
