# CLAUDE.md

## プロジェクト概要

サッカー・フットサルのスポーツイベント紹介用ランディングページを作成するプロジェクト。
シンプルな構成（HTML / CSS / JavaScript）で、見やすく魅力的なページを目指す。
heroes_web.md を設計書として、コーディングの参考にする

---

## よく使うコマンド

```bash
# ローカルサーバーで確認（Python）
python3 -m http.server 8080

# ローカルサーバーで確認（Node.js の live-server）
npx live-server

# ファイル構成を確認
ls -la

# HTMLの文法チェック（html-validate）
npx html-validate index.html
```

---

## デザイン方針

- サッカー・フットサルらしい **躍動感・エネルギー** を表現する
- 配色はチームカラーや大会テーマに合わせる（デフォルト: 緑・白・黒）
- モバイル閲覧を優先したレイアウト
- アニメーションは控えめに（パフォーマンス優先）
