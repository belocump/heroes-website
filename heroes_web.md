## 各セクション詳細

### ヘッダー（`.header`）

- 固定（`position: fixed`）、スクロール20px超で影が出る（`.scrolled`クラス）
- ロゴ：`HEROES` + `勿来店`（グリーン）
- ナビリンク：イベント / スタッフ / 特徴 / 今すぐ予約（ゴールドボタン）
- 900px以下でハンバーガーメニューに切り替わる
- モバイルナビはリンクタップで閉じる

### ヒーローセクション（`.hero` / `#top`）

- 全画面高さ（`100dvh`）、グリーングラデーション背景＋グリッドパターン
- ラベル：「屋内コートだから、雨でも絶対に開催。」（ゴールドボーダー）
- H1：「週末の充実感を、裏切らない。」
- サブテキスト：3種目・金土日参加可能
- CTAボタン2つ：「今週末のイベントを予約する」（グリーン）・「イベントを見る」（アウトライン）
- 下部にスクロールインジケーター（バウンスアニメーション）

### 屋内バナー（`.indoor-banner`）

ヒーローとイベントの間に配置。4アイテム横並び：
1. 雨天でも開催確定
2. スタッフが一緒にプレー
3. 金・土・日 開催
4. 1人参加でも安心

### イベントセクション（`.events` / `#events`）

- セクションヘッダー + TimeTreeカレンダー埋め込み（iframeで`https://timetreeapp.com/public_calendars/heroes_nakoso_2026/...`）
- 3カラムグリッド（900px以下で1カラム）
- カード3種：

| カード | バッジ | アイコン | ボタン色 |
|-------|-------|---------|---------|
| 大人の個サル | 金・土・日（グリーン） | ⚽ | グリーン |
| ソサイチ | 人気No.1 \| 土・日（ゴールド） | 🏆 | ゴールド |
| ウォーキングフットボール | 土・日（グリーン） | 🚶 | グリーン |

**注意：** 現在index.html内にソサイチカードが2枚重複している（バグ）。正しくは個サル・ソサイチ・ウォーキングフットボールの3枚。

各カードの構成：バッジ → アイコンエリア（グリーングラデーション背景）→ 本文（タイトル・キャッチコピー・説明・開催日・対象・予約ボタン）

### スタッフセクション（`.staff` / `#staff`）

- ダークグリーン背景・白テキスト
- スタッフカード（現在2枚・同一内容のプレースホルダー）
- 構成：アバターアイコン（👑）・担当イベント名（ゴールド）・引用コメント（ゴールド左ボーダー）・タグ3つ
- タグ例：「1人参加サポート」「ゲームメイク」「楽しい雰囲気作り」

### 特徴セクション（`.features` / `#features`）

3カラムグリッド（900px以下で2カラム、600px以下で1カラム）：
1. ⛈ 雨でも100%開催
2. 👥 スタッフが一緒にプレー
3. 🏕 1人参加が当たり前

### キャンセルポリシー（`.cancel-policy`）

- 白背景ボックス・ゴールド左ボーダー
- テキスト + 「予約ページへ」アウトラインボタン

### CTAセクション（`.cta` / `#reserve`）

- ダーク→ダークグリーングラデーション・中央揃え
- 「今週末のイベントを予約する（外部サイト）」ボタン（グリーン・XLサイズ）
- `id="reserve-btn"` で外部URL遷移処理が当たっている

### フッター（`.footer`）

- ほぼ黒背景
- ロゴ・ナビリンク（イベント/スタッフ/特徴/予約）・コピーライト

### フローティングボタン（`.floating-btn`）

- スマホのみ表示（600px以下で `display: block`）
- ヒーロー60%スクロール後〜CTAセクション手前で表示
- ゴールド背景・「今週末のイベントを予約する」
- `id="floating-btn"` で外部URL遷移処理が当たっている

## script.js の主要ロジック

| 機能 | 概要 |
|------|------|
| ヘッダー影 | スクロール量が20pxを超えたら `.scrolled` クラス付与 |
| ハンバーガーメニュー | クリックで `.open` トグル、リンクタップで閉じる |
| フローティングボタン表示制御 | ヒーロー高さ60%超＆CTAセクション手前でのみ表示 |
| スクロールアニメーション | IntersectionObserver で `.fade-up` → `.visible`（threshold: 0.12） |
| 時差アニメーション | イベントカード0.12s刻み、特徴アイテム0.1s刻み |
| スムーズスクロール | `a[href^="#"]` 全てに対してヘッダー高さ分オフセット |
| 予約ボタン外部遷移 | `RESERVE_URL` 変数（現在`'#'`）を設定すると外部タブで開く |
